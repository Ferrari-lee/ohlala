# 第三方登陆 #

支持 新浪微博、腾讯qq、google、github等第三方平台的登陆。

## 登陆流程 ##

```
0. 已经绑定过，自动登录 -> 4
1. 能拿到 email
  1.1 有email匹配的账户，自动绑定 -> 4
  1.2 没有匹配的 email 账户 -> 3
2. 无法拿到 email -> 3
3. 无法自动匹配
  3.1 用户选择登录并绑定 -> 4
  3.2 用户选择注册并绑定
    3.2.1 有 email -> 使用 email 创建新用户并绑定 -> 4
    3.2.2 无 email -> 输入 email 创建新用户并绑定 -> 4
4. 结束
```

## 编码设计 ##

面向接口编程，提供一个统一的接口，方便添加更多第三方登陆的支持。
